# Rock Paper Scissors Memory Game

## Description
This is a console-based game that combines the classic Rock Paper Scissors game with a memory challenge.

Players must memorize card positions within a limited time and use this information to compete against a computer opponent.

## Features
- Memory-based gameplay
- Multiple difficulty levels
- Timer system
- Score tracking
- Human vs Computer gameplay

## How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the program
